# Frontend Implementation Report - Exam Portal

## Project Overview
A comprehensive Online Examination Portal built with a modern React tech stack, featuring AI-driven proctoring and real-time monitoring.

## Tech Stack
- **Framework**: React with TypeScript
- **Styling**: Material UI (MUI)
- **State Management**: React Hooks (useState, useEffect, etc.)
- **API Interaction**: Axios
- **Real-time Communication**: Socket.io & WebRTC
- **AI Security**: face-api.js

---

## 1. Student Module

### Authenticiation & Entry
- **Login/Register**: Secure forms with validation.
- **Student Dashboard**: Overview of available and upcoming exams.
- **My Exams**: Detailed list of enrolled exams with status tracking.

### Examination Interface (`TakeExam.tsx`)
- **Fullscreen Enforcement**: Automatically enters fullscreen upon start.
- **Question Palette**: Interactive grid for quick navigation between questions.
- **Legend Indicators**: Color-coded status for Current, Answered, Marked for Review, and Unattempted questions.
- **Time Management**: Real-time ticker with visual warnings (Orange at 5 min, Red at 1 min).
- **Randomization**: Shuffles questions upon loading to prevent cheating.

### AI & Security Features
- **Face Detection (`CameraSecurity.tsx`)**: Periodic face monitoring using AI. Automatically warns or submits if the student leaves the camera view.
- **Tab Switiching Detection**: Monitoring `visibilitychange` and `focus` events. One warning is provided before automatic submission on the second attempt.
- **Security Checkpoint**: Entry-level face verification before starting the exam.

---

## 2. Teacher Module

### Management Dashboard
- **Teacher Dashboard**: At-a-glance stats and quick links to core features.
- **Manage Students**: view, edit, and monitor student profiles.
- **Manage Exams**: CRUD operations for exams, including viewing status and results.

### Exam Creation (`CreateExam.tsx`)
- **Multi-step Stepper**: Logical flow for Exam Details -> Questions -> Review.
- **Bulk Upload**: Support for Excel/CSV (XLSX) question imports.
- **Template System**: Ready-to-use CSV template for teachers.
- **Granular Controls**: Setting duration, passing marks, difficulty levels, and specific marks per question.

### Proctoring & Results
- **Live Proctoring (`LiveProctoring.tsx`)**:
    - Real-time video streams using **WebRTC**.
    - **Socket.io** signaling for peer connections.
    - Grid-based view of all active students.
- **View Results**: Detailed performance analytics and individual student responses.

---

## 3. Real-time Capabilities
- **WebRTC Integration**: Peer-to-peer video streaming from student camera to teacher dashboard.
- **Socket.io**: Persistent connection for room management (Join/Leave), WebRTC signaling (Offers/Answers/ICE), and live status updates.
